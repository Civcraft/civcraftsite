Citadel

Citadel is our protection plugin, designed to allow you to protect your buildings, doors, and chests from aggressors. It is built on the principle that protection requires resources, so to protect a block you must use any one of 3 resources: Smooth Stone, Iron, or Diamond. The more value the resource, the stronger the reinforcement.

PrisonPearl

PrisonPearl allows players to restrain other players (see this infographic for the basics). Keep in mind that every day you have a player pearled consumes 2 coal from the chest, if there is not enough coal the prisoners will be freed. Also PrisonPearl is integrated with CombatTag, logging out will not save you from being pearled.

Github

FactoryMod

Factorymod is a complex tech tree with the goal of allowing more efficient production from resources.. It transfers of all XP generation over to crop growing. You will not gain XP from killing mobs or mining normally. FactoryMod Wiki

Github

RealisticBiomes

RealisticBiomes is a mod to customize growth rates for various biomes to be more realistic as well as provide persistence in crop growth. See the guide here.

Github

ItemExchange

ItemExchange allows players to set up chests and dispensers as shops. It is used in conjunction with Citadel to lock the container that is being used this way. Any item can be set up to be traded for any item. A single shop can offer multiple exchanges, for customers to choose between. There's a guide here

Other plugins

Civpet allows you to un-tame animals for ownership transfer by feeding them cookies.

Dyncap lets us adjust the servers player cap on the fly for maximum player/performance ratio, the player cap is not actually 500 players and is subject to change while the server is live.

Arthropod Egg allows the use of a bane of arthropods enchanted weapon to occasionally cause mobs to drop corresponding spawn eggs. Higher level bane means higher chance of egg dropping.

MusterCull exists to limit the number of mobs for purposes of server performance. If you keep too many mobs in one spot (like a concentrated farm) MusterCull will target them for culling.
